# 视频字幕生成 Skill

为视频生成经过校对和优化的中文字幕视频。

## 前置依赖

运行前确认环境中已安装(缺失则提示用户安装):
- `ffmpeg`:音视频处理
- `yt-dlp`:YouTube 视频下载(当用户提供 YouTube 链接时需要)
- `whisper`(openai-whisper):语音识别

## 需要向用户确认的信息

1. **视频来源**:YouTube 链接或本地文件路径
2. **源语言**(必需,如 en / ja / ko;不确定时可让 Whisper 自动检测)
3. **视频背景 / 主题**(可选但强烈建议):用于翻译时纠正专有名词、
   人名、术语,以及消解语音识别的同音字错误。

## 执行步骤

### 步骤 0:下载视频(仅 YouTube 链接)

当用户提供 YouTube 链接时,先下载视频(优先 1080p,输出 mp4):

```bash
bash scripts/download.sh "<youtube_url>" "<output_dir>"
```

下载完成后,将输出的视频路径作为后续步骤的输入。

### 步骤 1:提取音频并生成原语言 SRT

调用脚本(参数:视频路径、源语言):

```bash
bash scripts/transcribe.sh "<video_path>" "<source_lang>"
```

它会在视频同目录生成 `<video_name>.orig.srt`。
执行后读取该 SRT 内容进入下一步。

### 步骤 2:AI 翻译与校对(由你——当前模型——完成)

读取 `<video_name>.orig.srt`,严格遵循以下规则输出中文 SRT:

- **保持 SRT 结构不变**:序号、时间轴(`00:00:01,000 --> 00:00:04,000`)
  必须原样保留,只翻译文本行。
- **结合背景校对**:利用用户提供的视频背景,纠正 Whisper 可能出现的
  同音字、专有名词、断句错误。若某句识别明显有误,先在心里还原正确
  原文,再翻译。
- **翻译优化**:输出地道、简洁、符合字幕阅读节奏的中文,避免生硬直译;
  单行过长时按语义合理拆分。专有名词/技术术语采用中文领域惯用译法,
  必要时保留英文原文(如首次出现的专有名词)。
- **不合并/不删除条目**:保持与原 SRT 一一对应的条目数量。

将结果写入 `<video_name>.zh.srt`。

## 完成后

向用户报告输出的中文 SRT 文件路径,并简要说明翻译中做过哪些明显的校对修正。
